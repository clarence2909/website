Important Note

This project is a frontend-only visual recreation. It does not copy backend logic, WordPress behavior, tracking, forms, integrations, or production data flows from the reference website.

Design Breakdown

Components: fixed header, mobile drawer, full-screen hero, centered brand carousel, expandable brand details, approach text grid, testimonial slider, recent-news cards, and large blue footer.

Layout: max-width 1440px container with 70px desktop gutters, 35px tablet/mobile gutters, large vertical pauses, centered hero typography, and rounded image surfaces.

Motion: fade-in hero, sticky header color transition, carousel scale/slide movement, expandable accordion-like brand panels, card hover zoom, and scroll reveal fades.

Typography and color: light editorial sans-serif styling, black body type, pale blue #BEE1F7, burgundy #6D2D33, white, and soft neutral card backgrounds.

Responsiveness: desktop uses horizontal carousel and multi-column sections; tablet tightens typography and spacing; mobile switches to drawer navigation, stacked brand cards, and single-column content.
